# Incredibot

## Nécessaire:
* Python >=3.7 <3.10 (je conseille la dernière version de 3.9 (ajd 3.9.10))
* les trucs de base (pip, setuptools, docutils ...) à jour
* kivy[base] , si qqun a la fois de ts les écrire, sinon pip -m install kivy[base] ça marche

* Etage 1
