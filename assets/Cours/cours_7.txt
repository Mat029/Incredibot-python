En vous ayant introduit les bases des bases de l’informatique moderne, nous allons maintenant essayer de les completer.
Tout d’abord, dans un premier temps, un programme peut contenir des affectations de variables.
Une variable est un symbole dans lequel il est possible de stocker une valeur. L’affectation de variable est l’action de stocker une valeur dans une variable.
Par exemple, si vous voulez compter le nombre de tour que fait une machine à laver vous aller d’abord definir votre variable à 0
nbtour → 0 
(j’ai volontairement miss une flèche ici car elle depend du language utilisé)
Puis dès que vos capteurs se rencontrent, vous aller ajouter 1 à votre variable :
nbtour → nbtour + 1
C’est a peu près ce qui ce passe dans les niveaux avec des objets.
En prenant un objet, la valeur 1 va être affecetrer à la variable charger de savoir si le robot à pris l’objet ou non ( en l’occurence AGATHE 0 COMPLETER dans notre code).
