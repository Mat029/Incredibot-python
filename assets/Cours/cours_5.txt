[b] Introduction à la programmation : [/b]

Les premiers cours étaient des tutoriels pour le jeu. Dans cette partie nous allons vous donner quelques notions fondamentales sur la programmation.
Tout d’abord, qu’est-ce que programmer ?
Programmer consiste à mettre des instructions de façon ordonnée (comme vous le faites pour le robot) de façon à ce qu’elles produisent quelque chose lorsqu’on les exécute. L'ensemble des instructions s'appelle un programme. C’est de là que vient le terme programmer, c’est-à-dire créer un programme.
Dans Incredibot, sans le savoir, vous avez écrit plusieurs programmes (Un programme « Incredibot » car les instructions “avancer” et “reculer” n’ont de signification que dans Incredibot) qui ont été transmis au robot qui a agi en suivant vos instructions.
Il en va de même si vous voulez programmer une application, un site web, ou une intelligence artificielle (même si là ça devient assez complexe). Dans tous les cas vous allez ajouter des instructions pour obtenir un résultat. On écrit ces instructions dans un langage spécial qu’on appelle un langage de programmation. Il en existe de nombreux (C++, Java, Python etc.…) qui permettent chacun de faire des choses différentes. Ce langage est ensuite transformé en instructions compréhensibles par l’ordinateur à travers un procédé complexe que nous ne pouvons pas expliquer ici (si jamais cela vous intéresse, taper comment fonctionne un langage de programmation sur un moteur de recherche, vous trouverez plein de choses intéressantes). Pour le moment, le seul langage que vous avez utilisé est le langage Incredibot qui n’est utile que dans notre application. Notre application, elle, est codé dans un langage très populaire appelé le python.
Dans les prochains chapitres, nous vous introduirons les concepts principaux de la programmation pour que vous puissiez vous aussi commencer à apprendre le Python !
